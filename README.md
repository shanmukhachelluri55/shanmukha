# Shanmukha Chelluri - Portfolio Website

A personal portfolio website for Shanmukha Chelluri, showcasing skills, experience, projects, and education.

## Features

- Responsive design for all devices
- Interactive background animation
- Dark/light mode toggle
- Smooth scrolling navigation
- Animated skill sections
- Timeline for work experience
- Project showcase
- Contact form with validation
- Social media links

## Technologies Used

- HTML5
- CSS3 (with Tailwind CSS)
- JavaScript (Vanilla JS)
- Font Awesome for icons
- Google Fonts

## Project Structure

```
├── index.html            # Main HTML file
├── css/
│   ├── tailwind.css      # Tailwind CSS configuration
│   └── styles.css        # Custom CSS styles
├── js/
│   ├── particles.js      # Background animation
│   └── main.js           # Main JavaScript functionality
└── README.md            # Project documentation
```

## Setup and Installation

1. Clone the repository:
   ```
   git clone https://github.com/shanmukhachelluri55/portfolio.git
   ```

2. Open the project folder:
   ```
   cd portfolio
   ```

3. Open `index.html` in your browser or use a local server.

## Customization

- **Personal Information**: Update your name, contact details, and social media links in `index.html`.
- **Experience**: Modify the work experience section in `index.html`.
- **Projects**: Add or remove projects in the projects section.
- **Skills**: Adjust the skills displayed in the skills section.
- **Colors**: Customize the color scheme in `tailwind.config.js`.

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
